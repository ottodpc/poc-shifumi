<h2>🛠️ Installation Steps:</h2>

<p>1. clone the project</p>

```
git clone git@github.com:ottodpc/poc-shifumi.git
```

<p>2. install deps</p>

```
 cd ./poc-shifumi && yarn
```

<p>3. start the project</p>

```
yarn start
```
